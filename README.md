# freelancer
This project was made to practice HTML and CSS with Responsive Web Design using Flexbox and Grid. It is a website for a freelance developer where there is a section my services and a contact form.

## Tools used:
- Normalize CSS
- Tabler Icons
- Google Fonts
- ResponsivelyApp
